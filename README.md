# Eloise-agro

Projeto frontend para o Concurso Sakura de Agronegócios — interface responsiva e visual com foco educativo.

Como visualizar localmente:

1. Abra o arquivo [index.html](index.html) no seu navegador.
2. Para desenvolvimento com refresh automático, use uma extensão Live Server ou rode um servidor simples:

```
# Python 3
python3 -m http.server 8000

# então abra http://localhost:8000
```

Arquivos principais:

- [index.html](index.html) — estrutura semântica e formulário de inscrição.
- [css/styles.css](css/styles.css) — estilos com variáveis, Grid/Flexbox e animações.
- [js/main.js](js/main.js) — validação básica do formulário.

Próximos passos sugeridos: adicionar backend para persistência das inscrições, testes de acessibilidade e otimizações de imagens.
